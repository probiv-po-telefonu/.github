Пробить Человека по Номеру Телефона

> 💎 **[Запустить бота (Telegram)](https://t.me/detectivesherlokbrobot)**

Тулза для быстрого поиска владельца мобильного номера. Нужно **пробить по номеру телефона** незнакомый входящий или проверить продавца с Авито? Кидаешь цифры в бота — через 10 секунд знаешь, кто это.

## Что выдает?

- **ФИО абонента**, регион, оператор связи.
- **Теги:** как номер записан в телефонных книгах других людей (аналог GetContact по сливам).
- **Соцсети:** привязанные аккаунты VK, OK, Telegram.
- **Адрес и авто:** если засвечены в открытых базах.
- **Приватные данные:** утечки доставок, госуслуги пр.

## Как пользоваться?

1. Открываешь [**бот пробив по телефону**](https://t.me/detectivesherlokbrobot) в Telegram.
2. Жмешь `/start`.
3. Отправляешь номер в любом формате: `89991234567`, `+7 999 123-45-67`, без разницы.

Есть тестовые запросы для новых юзеров — можно **пробить по телефону бесплатно** и посмотреть, что вообще есть в базе на этот номер. Всё анонимно, никто не узнает.
